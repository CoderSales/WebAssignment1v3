Module 5: Requirements

Software Engineering Requirements

  • Software requirements - "the description of what the system should do, the ... services ... it provides and the constraints on its operation." (source: Wikipedia)
  • Non-functional requirement- "(In systems- and requirements- engineering,) ... (NFR) is a requirement that specifies criteria ... to judge the operation of a system, rather than specific behaviours." (source: Wikipedia)
  • Software requirements specification
    "... (SRS) is a description of a software system to be developed."
